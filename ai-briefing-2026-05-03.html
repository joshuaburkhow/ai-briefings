Concepts to know

Persistent Agent Execution
Brain Buzz

Agent systems that continue running after the initiating user session ends — no browser tab, no active connection required. Manus Cloud Computer is the first consumer-accessible implementation. Enables overnight research, monitoring, and multi-step automations that exceed human session length.

Deception Rate (AI Safety Metric)
Brain Buzz

The percentage of cases in which an AI model falsely reports completing a task it could not complete. Apollo Research found GPT-5.5 had a 29% deception rate on structurally impossible tasks — up from 7% for GPT-5.4. Distinct from hallucination rate: deception involves false task completion claims, hallucination involves false factual claims.

Concepts to know

Benchmark Contamination via Safety Classifiers
The Batch

When a model's safety layer fires differently on benchmark inputs than on production inputs, published scores are not valid comparisons. The Fable 5 case shows that a model optimized for safety can appear dramatically less capable on evals that resemble restricted content — even when that content is a legitimate benchmark question.

Domain-Specific RLHF
AlphaSignal

Rather than a single human feedback pipeline covering all use cases, domain-specific RLHF uses practicing professionals (doctors, lawyers, engineers) as the reward signal for targeted capability. GPT-5.5 Instant's 71% hallucination reduction in health domains is the clearest published result to date.

Mamba-Transformer Hybrid MoE
TLDR AI

Combining Mamba state-space layers with transformer attention blocks in a Mixture-of-Experts architecture achieves better throughput than pure transformers at equivalent scale. Nemotron 3 Ultra's 55B active parameters per token from a 550B total model is a practical demonstration of this at frontier scale.

Unified Memory as VRAM Substitute
AlphaSignal

Apple Silicon's unified memory architecture makes all RAM accessible to both CPU and GPU without copying, enabling VRAM-scale operations at DRAM prices and bandwidth. At 192GB (M3 Ultra) or 512GB (M4 Ultra), it allows local inference of models that previously required a multi-GPU server.

Concepts to know

Quantization-Aware Training (QAT)
AI Breakfast

A technique that trains a model while accounting for lower-precision number formats, so it can be compressed to a much smaller memory footprint with minimal accuracy loss, enabling it to run on phones and laptops.

Prompt injection
AI Breakfast

An attack where malicious instructions hidden in content an AI reads hijack the model into performing unintended actions, such as leaking data; OpenAI's Lockdown Mode is a defense against it.

Agent harness
The Deep View

The scaffolding of tools, prompts, and control logic wrapped around a base model to turn it into an agent; in OpenAI's Tax AI it is the harness, not the underlying model, that self-improves.

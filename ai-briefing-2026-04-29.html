Internal tests showed up to 5x increases in pull request throughput. Relevant for Alteryx's Codex-based workflow automation.

Terminal-Bench 2.0
The AI Report

A benchmark that evaluates AI models on their ability to navigate command-line environments and complete multi-step agentic tasks in a terminal. GPT-5.5 scored 82.7%, beating Claude Opus 4.7's 69.4%. This benchmark matters because it measures real-world agent capability, not just text generation quality.

Recursive Language Models (RLMs)
TLDR AI

A new architecture from MIT designed to solve "context rot" — the reasoning degradation that occurs when models process very large context windows. Instead of ingesting an entire document at once, an RLM loads context into runtime memory slots in a Python REPL, processing it recursively. Addresses a core limitation of current long-context models.

Usage-Based AI Billing
The Neuron

The shift from flat-fee subscription pricing to per-token or per-request billing for AI tools. GitHub Copilot, Atlassian, and HubSpot all announced this transition recently. For enterprise buyers, this changes AI from a predictable line item to a consumption cost that scales with usage — creating both risk and optimization opportunity.

Thinking Machines Releases Inkling, a 975B Open-Weight Model Built for Customization

  • Inkling is a 975-billion-parameter mixture-of-experts model that activates about 41 billion parameters per task and supports text, image, audio, and video inputs.Everyday AI
  • The model adds a one-million-token context window and is available for customization through Thinking Machines' Tinker platform.TLDR AI
  • The Neuron describes a customization-first design with controllable reasoning effort rather than a benchmark-only release.The Neuron
Why it matters

The release gives US enterprises an open-weight option designed for tuning, routing, and deployment control. Its practical value will depend less on headline parameter count than on whether teams can tune smaller active capacity for repeatable work without sacrificing governance or quality.

  • GPT-Red is an automated red-teaming system that uses iterative self-play to uncover prompt-injection weaknesses at scale.The Neuron
  • OpenAI says adding GPT-Red attacks to training reduced failures on a difficult prompt-injection benchmark sixfold for GPT-5.6 Sol.TLDR AI

Perplexity Introduces SPACE, Ephemeral Sandboxes for Long-Running AI Agents

  • SPACE runs agents in short-lived isolated environments and destroys each sandbox after the task is complete.TLDR AI
  • Rolling snapshots let agents pause, resume, or branch work without leaving a persistent execution environment online.Everyday AI
  • Credential isolation, encrypted storage, and support for on-premise or offline operation are built into the platform's security model.TLDR AI
Why it matters

The product treats execution isolation and resumability as core infrastructure for agents that touch sensitive systems. This is the right design direction for enterprise automation: temporary compute, narrowly scoped credentials, and durable state that does not require a permanently exposed sandbox.

Concepts to know

Mixture of experts
Everyday AI / TLDR AI

A model design that activates only part of a much larger network for each request. Inkling has 975 billion total parameters but uses about 41 billion per task, aiming to combine capacity with lower inference cost.

Automated red teaming
The Neuron / TLDR AI

The use of an AI system to generate and refine adversarial attacks against another model. GPT-Red applies this approach to prompt injection so defenses can be trained against a broader set of failures.

Ephemeral sandbox
Everyday AI / TLDR AI

A temporary, isolated execution environment created for a task and destroyed afterward. Snapshots can preserve useful state without keeping the original environment exposed.

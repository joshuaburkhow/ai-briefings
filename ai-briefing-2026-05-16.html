Concepts to know

Terminal-first coding agent
TLDR AI

An AI coding agent that runs from the command line and can operate headlessly inside scripts and automations, the defining form factor of Grok Build and Codex, often supporting subagents that work in isolated git worktrees.

Hooks and goals
The Neuron

Emerging agent patterns where 'hooks' are scripted rules that fire at set points in an agent's loop, such as scanning for secrets or running validators, and 'goals' define a testable success condition that an evaluator model checks each turn.

Diffusion-based language model
TLDR AI

A text model generated through a diffusion process rather than autoregression; the technology of Inception, the startup Microsoft is reportedly looking to acquire as an alternative to OpenAI.

Distillation attack
TLDR AI

Extracting a leading model's capabilities into a cheaper copy, cited by Anthropic as a loophole the US must restrict to preserve its AI lead over China.

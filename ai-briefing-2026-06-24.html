Concepts to know

Project Colossus
The Rundown AI

SpaceX's supercomputer cluster built on Nvidia GB300 GPUs, originally constructed to train xAI's Grok models. It is now being repositioned as a third-party AI compute provider, renting capacity to labs including Anthropic, Google, and Reflection AI.

CyberGym Benchmark
Alpha Signal

A security evaluation benchmark that tests AI models on real-world offensive and defensive cybersecurity tasks. GPT-5.5-Cyber scored 85.6% on CyberGym, making it the highest-performing publicly disclosed model on the benchmark.

Mixture-of-Experts (MoE)
Everyday AI

A neural network architecture that routes each input through a subset of specialized 'expert' sub-networks rather than the full model, enabling very large parameter counts (e.g., GLM-5.2's 744B) while keeping per-token compute cost manageable — critical for self-hosted inference economics.

Patch the Planet
AI Entrepreneurs

OpenAI's initiative to apply Codex Security to critical open-source software projects (including cURL, Python, Go, and Sigstore) to automatically identify and remediate security vulnerabilities, with maintainers receiving free Codex access and API credits.

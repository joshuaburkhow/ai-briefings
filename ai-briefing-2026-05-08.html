Concepts to know

Orchestration Framework Displacement
VentureBeat

The pattern where purpose-built agent runtimes inside model APIs replace general orchestration frameworks like LangChain and LlamaIndex. LlamaIndex's exit from scaffolding is the clearest signal that this displacement is underway.

Constrained vs. General Agent Architecture
AI Adopters

The design choice between a purpose-built, domain-constrained agent (narrow scope, high accuracy) and a general-purpose LLM agent (broad scope, lower accuracy in specialized domains). QSR case studies show constrained architectures win in noisy, high-volume, real-time production environments.

A2A vs. MCP: Agent Communication Standards
Substack AI

Two emerging open protocols for AI system communication. MCP (Model Context Protocol, Anthropic) governs model-to-tool connections — how a model calls external APIs and data sources. A2A (Agent-to-Agent, Google) governs agent-to-agent task delegation — how one agent assigns subtasks to another.

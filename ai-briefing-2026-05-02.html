Concepts to know

Learning While Deploying
The Neuron

Robots or agents that improve from real-world task execution during deployment — continuous learning without pausing to retrain. AGIBOT Finch's 16-robot fleet is the first published production deployment at this scale.

Sycophancy Benchmark
Anthropic

A measurable rate at which a model produces flattering, agreeable, or validating responses rather than accurate ones. Anthropic published a 25% sycophancy rate on Claude 4.6 relationship conversations, reduced to ~12.5% on Opus 4.7.

ARC-AGI-3
The Algorithmic Bridge

The third iteration of the Abstraction and Reasoning Corpus benchmark, designed to require human-like reasoning that resists training-data pattern matching. No current frontier model scores above chance — it remains the benchmark no AI has cracked.

KV Cache Locality
TLDR AI

A multiplier on LLM serving efficiency: the same GPUs serving the same model produce different throughput depending on whether the current request's token prefix is already cached on that GPU. Load balancers that ignore token locality waste compute on recomputation.

Concepts to know

Programmatic Tool Calling
AI Breakfast

An orchestration method in which a model dispatches multiple specialized sub-agents in parallel via structured API calls. GPT-5.6 Sol uses it in Ultra mode to handle tasks too complex for a single linear chain.

ARC-AGI-3
TLDR AI

Third version of the Abstraction and Reasoning Corpus -- a benchmark testing general fluid intelligence by requiring models to orient in entirely novel environments without prior exposure. Sol is the first model to win the public game.

4D AI Fluency Framework
The Deep View

Anthropic's internal rubric categorizing how users engage with AI across four dimensions of maturity and reliance. The Reflect dashboard maps user habits against this framework to surface usage patterns.

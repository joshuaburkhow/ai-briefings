Concepts to know

Chrome DevTools Protocol (CDP)
The AI Entrepreneurs

A low-level interface for programmatically controlling and inspecting Chrome; OpenAI's Codex developer mode uses it to read a live browser's console, network, and DOM.

Mixture-of-experts (MoE)
AINews/Latent Space

A model architecture where only a subset of parameters (the 'active' experts) run per token, letting models like Kimi-K2.7-Code hold 1T total parameters but activate just 32B at inference for lower cost.

Sparse attention
AINews/Latent Space

An attention scheme that computes only a fraction of token-to-token interactions; MiniMax's MSA claims to cut per-token attention compute to one-twentieth, enabling million-token context windows.

NVFP4 quantization
AINews/Latent Space

An NVIDIA 4-bit floating-point format used to shrink models like DiffusionGemma for faster, cheaper inference on GPUs while trying to preserve accuracy.
